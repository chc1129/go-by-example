サンプルで学ぶ Go 言語：Recursion

Go は再帰関数をサポートしている。 この例は古典的な階乗を計算するものだ。

 
package main

import "fmt"

この関数 fact は、ベースケースである fact(0) に到達するまで自身を繰り返し呼び出す。

 
func fact(n int) int {
    if n == 0 {
        return 1
    }
    return n * fact(n-1)
}

func main() {
    fmt.Println(fact(7))
}

$ go run recursion.go 
5040
